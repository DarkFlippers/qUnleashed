qUnleashed - Flutter-приложение-компаньон для Flipper Zero, ориентированное на работу с кастомными прошивками, в первую очередь Unleashed firmware.

Приложение поддерживает подключение по BLE, отслеживание релизов прошивки, просмотр changelog, загрузку обновлений, установку прошивки из приложения, просмотр информации об устройстве, управление архивом, файловый браузер, трансляцию экрана, удаленное управление, CLI-доступ, каталог приложений, восстановление MIFARE Classic ключей через Mfkey32, инфракрасную библиотеку и FlipperMap для координат из синхронизированных файлов.

Проект является свободным ПО и разрабатывается из исходного кода в публичном репозитории.
